# Security Policy

## Supported Versions

Security fixes target the latest npm release and the current `main` branch. Older `0.0.x` releases may not receive backports.

## Reporting

Please report suspected vulnerabilities privately to `WATT3D@protonmail.com`. Include affected versions, reproduction steps, impact, and any suggested mitigation. Avoid opening a public issue until the maintainer has had a reasonable opportunity to investigate and coordinate a fix.

This address is for security reports, not general support or licensing requests.
